install ruby

```sh
sudo apt-get install ruby1.9.3
```

install jekyll

```ruby
gem install jekyll
```

use rdiscount

```ruby
gem install rdiscount
```

edit ` __config.yml`

```markdown
markdown: rdiscount
```
